## 🔧 Using the Project

### Run the CLI
```bash
cd task_manager
python -m src.cli
```

### Run Tests
```bash
pytest tests/
# or
python -m unittest discover tests/
```

### Use in Python
```python
from src import TaskManager, TaskPriority

manager = TaskManager()
task = manager.add_task("My Task", "Description", TaskPriority.HIGH)
print(manager.get_pending_tasks())
```

### Save/Load Tasks
```python
manager.save_to_file("my_tasks.json")
new_manager = TaskManager()
new_manager.load_from_file("my_tasks.json")
```
